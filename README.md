# BestZappy

## Server

### Install

Please install the dependencies and compile the library :

```bash
apt install g++ qt5-default -y
cd server/<build-directory>
make
```

### Run

```bash
./server
```

## Client

### Install

```bash
apt install g++ qt5-default -y
cd client/build
make
```

### Run

```bash
cd client/build
./client --host <host> --port <port>
```
